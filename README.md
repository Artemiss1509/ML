### 1. **Cat vs Dog Image Classifier (CNN)** 
**File:** `fcc_cat_dog.ipynb`

A deep learning image classification project using Convolutional Neural Networks (CNN) to distinguish between cats and dogs. 

**Key Features:**
- Built with TensorFlow/Keras
- CNN architecture with multiple convolutional and pooling layers
- Image preprocessing and augmentation
- Model training and validation
- Performance evaluation with accuracy metrics

**Technologies:** TensorFlow, Keras, CNN, Computer Vision, Image Classification

---

### 2. **Book Recommendation System (k-NN)**
**File:** `fcc_book_recommendation_knn.ipynb`

A collaborative filtering-based book recommender system using k-Nearest Neighbors algorithm.

**Key Features:**
- k-NN algorithm implementation for recommendations
- User-item interaction matrix
- Similarity-based filtering
- Personalized book recommendations
- Evaluation metrics for recommendation quality

**Technologies:** scikit-learn, k-NN, Collaborative Filtering, Recommender Systems

---

### 3. **SMS Spam Detector (NLP)**
**File:** `sms_text_classification.ipynb`

A natural language processing project to classify SMS messages as spam or legitimate (ham).

**Key Features:**
- Text preprocessing and tokenization
- Feature extraction (TF-IDF/Count Vectorization)
- Binary classification model
- NLP techniques for text analysis
- Model evaluation with precision, recall, and F1-score

**Technologies:** NLP, Text Classification, scikit-learn, TensorFlow

---

### 4. **Health Cost Prediction (Regression)**
**File:** `fcc_predict_health_costs_with_regression.ipynb`

A regression model to predict healthcare costs based on patient characteristics.

**Key Features:**
- Linear/Neural network regression
- Feature engineering and normalization
- Predictive modeling for cost estimation
- MAE (Mean Absolute Error) evaluation
- Data visualization and analysis

**Technologies:** TensorFlow, Regression Analysis, Predictive Modeling

---

### 5. **Rock Paper Scissors AI (Pattern Recognition)**
**File:** `RPS.py`

An AI player that uses pattern recognition and sequence prediction to beat opponents in Rock Paper Scissors.

**Key Features:**
- Pattern recognition algorithm
- Opponent move history tracking
- Sequence-based prediction (stride method)
- Frequency analysis for next-move prediction
- Win rate optimization

**Technologies:** Python, Pattern Recognition, Game AI
